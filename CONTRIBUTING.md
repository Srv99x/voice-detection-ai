# Contributing to AI Voice Detector

Thank you for your interest in contributing! This project was built for a CyberSecurity Hackathon and is actively maintained. All contributions are welcome — from bug fixes to new features.

---

## 🚀 Getting Started

1. **Fork** the repository on GitHub
2. **Clone** your fork locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/voice-detection-ai.git
   cd voice-detection-ai
   ```
3. **Create a branch** for your change:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Set up** your environment (see [README.md](README.md#local-setup))
5. **Make your changes**, commit, and push
6. **Open a Pull Request** against the `main` branch

---

## 🧭 Areas to Contribute

| Area | Description |
|------|-------------|
| 🎯 **Model Accuracy** | Improve detection with more training data or a better classifier |
| 🌍 **Language Support** | Improve detection on non-English audio |
| 📦 **Audio Formats** | Add support for OGG, FLAC, M4A |
| 🌊 **Streaming Input** | Support real-time audio stream detection |
| 🧪 **Tests** | Expand `test_api.py` coverage |
| 📚 **Docs** | Improve README, add examples, fix typos |
| 🎨 **Frontend** | Improve the React UI in the `frontend/` folder |

---

## 📋 Pull Request Guidelines

- Keep PRs focused — one feature or fix per PR
- Write clear commit messages: `fix: correct JSON field ordering` or `feat: add OGG audio support`
- If adding a new feature, update the README accordingly
- If changing the model pipeline, include before/after accuracy metrics

---

## 🐛 Reporting Bugs

Open an [Issue](https://github.com/Srv99x/voice-detection-ai/issues) with:
- A clear description of the bug
- Steps to reproduce it
- Expected vs actual behavior
- Your Python version and OS

---

## 💡 Suggesting Features

Open an [Issue](https://github.com/Srv99x/voice-detection-ai/issues) with the `enhancement` label and describe:
- The problem you want to solve
- Your proposed solution
- Any relevant examples or references

---

## 🤝 Code of Conduct

Be respectful and constructive. This is a learning project — all skill levels are welcome.
